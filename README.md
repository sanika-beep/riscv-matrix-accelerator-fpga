# RISC-V Matrix Accelerator FPGA

A 32-bit pipelined RISC-V processor with a custom matrix multiplication accelerator implemented in Verilog and targeted for Xilinx Spartan-7 FPGA.

## Project Goals

- RV32I Processor
- 5-stage Pipeline
- Hazard Detection Unit
- Data Forwarding Unit
- Matrix Multiplication Accelerator
- FPGA Implementation

## Folder Structure

rtl/    -> Verilog source files

tb/     -> Testbenches

docs/   -> Architecture diagrams and reports

fpga/   -> Constraints and FPGA files
